// WHAT IS MMA?
MULTIMEDIA AND AUTHORING

Multimedia and Authoring (MMA) is a core subject in the BSIT curriculum at TUP — Taguig. It covers the integration of digital media — sound, video, animation, graphics — and teaches students to author interactive multimedia content using industry-standard tools.

This portfolio project merges MMA principles with creative exploration: using 3D rendering in Blender to visualize the solar system, and rotoscoping in FlipAClip to animate planets through hand-traced frame sequences.

// 12-WEEK JOURNEY
PROJECT TIMELINE

Weeks 1–3: Foundation

Learning rotoscoping basics, setting up Blender environment, and establishing animation style for the planets.

Weeks 4–6: Inner Planets

Mercury, Venus, Earth, Mars — detailed 3D renders and frame-by-frame rotoscoping sequences completed.

Weeks 7–9: Gas Giants

Jupiter and Saturn with ring system, complex procedural shaders in Blender using Python automation.

Weeks 10–12: Final Output

Uranus, Neptune, composite sequence, final 60-second animation assembly and portfolio polish.

// CURRICULUM CONNECTION

MMA × Rotoscoping × 3D

01
DIGITAL MEDIA AUTHORING

MMA teaches students to combine multiple media types. This project integrates 3D animation, video, and interactive web presentation — the full spectrum of multimedia authoring.

02
ROTOSCOPING TECHNIQUE

Rotoscoping traces over existing footage frame-by-frame. Applied here to planetary motion — blending real astronomical movement with hand-crafted animation using FlipAClip.

03
3D PIPELINE INTEGRATION

Blender with Python scripting automates render passes, camera animations, and shader generation — showing how IT programming skills power creative media production.
